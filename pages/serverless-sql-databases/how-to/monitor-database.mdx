Scaleway Serverless SQL Databases is fully integrated into [Cockpit](/cockpit/quickstart/), and allows you to quickly access your logs and metrics. This page explains how to use Scaleway Cockpit to monitor your Serverless SQL Databases.

<Requirements />

- A Scaleway account logged into the [console](https://console.scaleway.com)
- [Owner](/iam/concepts/#owner) status or [IAM permissions](/iam/concepts/#permission) allowing you to perform actions in the intended Organization
- [Created a Serverless SQL Database](/serverless-sql-databases/how-to/create-a-database/)

## How to access the Grafana dashboard

2. From the Scaleway console [Cockpit overview page](https://console.scaleway.com/cockpit/overview), click **Access Grafana** in the top right corner.

4. From the **Scaleway Dashboards** panel, select **Serverless SQL Databases Overview** to access the metrics of your databases.

  You are directed to the selected dashboard. You can use the filters at the top of the page to refine the displayed information.

<Message type="note">
  You can also access your Serverless SQL Databases metrics from the [console](https://console.scaleway.com) via the **Metrics** tabs of a database.
</Message>

## Available metrics

The following metrics are available to monitor your Serverless SQL Databases:

- Provisioned CPU
- CPU usage
- Provisioned memory
- Memory usage
- Client connections
- Size of the data stored
- Disk usage (read / write)
