* The **Public bandwidth** refers to the designated internet bandwidth for the server, while the **Private bandwidth** signifies the permitted bandwidth for the server within Private Networks.
  Both public internet and Private Networks use the same physical network interface.
* **Water usage** is only included for zones where values are available; missing zones indicate unavailable data.
</Message>
