You can reset your Grafana password using the following request:

```bash
curl --location --request POST 'https://api.scaleway.com/cockpit/v1beta1/reset-cockpit-grafana-password' \
--header 'Content-Type: application/json' \
--header 'Accept: application/json' \
--header "X-Auth-Token: ${SCW_API_KEY}" \
--data-raw '{}'
```

The output displays a new password for your user.

<Message type="important">
 The username is always the same. You can get it on [the Scaleway console](https://console.scaleway.com/cockpit/users).
</Message>
