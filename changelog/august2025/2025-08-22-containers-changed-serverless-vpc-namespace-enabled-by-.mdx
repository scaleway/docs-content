---
title: Serverless VPC namespace enabled by default
status: changed
date: 2025-08-22
category: serverless
product: containers
---

To be able to attach Serverless Containers to a Private Network, previously it was available only for namespaces with `VPC Enabled` feature. Now it's available on all Serverless Containers Namespaces.

[How to use Private Networks with Serverless Containers](https://www.scaleway.com/en/docs/serverless-containers/how-to/use-private-networks/)
