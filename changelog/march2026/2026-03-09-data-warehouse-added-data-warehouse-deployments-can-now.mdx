---
title: Data Warehouse deployments can now be stopped
status: added
date: 2026-03-09
category: data-and-analytics
product: data-warehouse
---

It is now possible to stop Data Warehouse deployments.
When a deployment is stopped, you will no longer be billed for vCPU consumption - only for Object Storage. This allows for better cost management, and you can start the deployment again at any time.
Refer to our [documentation](/data-warehouse/how-to/start-stop-deployment/) for more information.
