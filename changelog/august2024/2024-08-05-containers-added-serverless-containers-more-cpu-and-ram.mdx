Serverless Containers now support higher CPU and RAM limits, allowing for more demanding workloads. This update provides more power and flexibility to meet your application's needs.

In addition, we have introduced a balanced resource allocation system, establishing a relationship between RAM and CPU to help you select optimal resource combinations.
