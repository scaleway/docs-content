Serverless Functions are billed on a pay-as-you-go basis depending on:

* Monthly Request number
* Resource consumption
* Resources provision

Refer to the [Serverless Functions FAQ](/faq/serverless-functions/#how-am-i-billed-for-serverless-functions) for more information on billing.
