---
title: New minimum durations for lifecycle rules transitions
status: changed
date: 2026-03-31
category: storage
product: object-storage
---

Scaleway Object Storage now has a minimum duration for lifecycle transitions, for predictable behavior at scale:
- 90 days minimum before transitioning objects to Glacier
- 30 days minimum before transitioning objects to Standard One Zone

Existing lifecycle rules below these limits will continue to apply, but we recommend updating existing rules to comply with these new durations for optimal lifecycle management. Any update to an existing lifecycle rule will require compliance with these minimum durations.
