---
title: Data Warehouse deployments can now be stopped
status: added
date: 2026-03-09
category: data-and-analytics
product: data-warehouse
---

It is now possible to stop a Data Warehouse deployment.
When a deployment is stopped, you will only be billed for Object Storage, which allows you to do cost management. You can then start the deployment again at any time.
Refer to the [documentation](https://www.scaleway.com/en/docs/data-warehouse/how-to/start-stop-deployment/) for more information.
