## What is Object Storage? Quickstart tutorial with use cases


Object Storage allows you to store different types of objects (documents, images, videos, etc) and to distribute them instantly, anywhere in the world. We explain some core Object Storage concepts, take you step by step through the creation of an Object Storage bucket in the cloud, and tell you about some common use cases. You'll learn how to store and share files, how to host a static website from a bucket, and more.
