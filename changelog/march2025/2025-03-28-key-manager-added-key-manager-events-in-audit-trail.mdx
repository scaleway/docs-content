### What is included

You can now track encryption key creation, editing, rotation and deletion. Find out more about the Key Manager supported Audit Trail endpoints in the [dedicated documentation](/audit-trail/reference-content/adt-supported-endpoints/#key-manager).

### Key benefits
- Full visibility within 2 seconds of action completion
- Complete audit logs with user IDs, timestamps
- Simple filtering by action type and principal
- One-month log retention

Access this feature in the [Audit Trail section](https://account.scaleway.com/?service=elementsaudit-trail/) of the Scaleway console.
