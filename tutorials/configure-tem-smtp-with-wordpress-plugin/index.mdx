- A Scaleway account logged into the [Scaleway console](https://console.scaleway.com)
- [Owner](/identity-and-access-management/iam/concepts/#owner) status or [IAM permissions](/identity-and-access-management/iam/concepts/#permission) allowing you to perform actions in the intended Organization
- A valid [API key](/identity-and-access-management/iam/how-to/create-api-keys/) with the right [permissions](/identity-and-access-management/iam/reference-content/permission-sets/)
- [Configured your domain with Transactional Email](/managed-services/transactional-email/how-to/configure-domain-with-transactional-email/)
- [Added SPF, DKIM](/managed-services/transactional-email/how-to/add-spf-dkim-records-to-your-domain/), and [MX](/managed-services/transactional-email/how-to/add-mx-record-to-your-domain/) records to your domain

## Creating your resources

1. Créer une clé API et bien noter la secret key car c’est le mdp 
2. Créer une Instance avec une image wordpress 
3. Tabs > Security group
4. Créer un security group, Autoriser SMTP port
5. Copier/coller l’IP publique de l’Instance sur le navigateur
6. Créer son site (nom du site, utilisateur, mdp et adresse mail)


## Se connecter à son site WordPress et installer le Plugin

1. Se connecter à son site avec le mdp et utilisateur que tu viens de créer  
2. Cliquer sur « Plugins » dans le menu de gauche. Normalement tu arrives dans « Installed Plugins ». 
3. Si le plugin n’est pas dans « Installed Plugins » > Cliquer sur « Add New Plugin ». 
4. Dans la barre de recherche, taper « WP Mail SMTP » et cliquer « Install Now »
5. Aller dans « Installed Plugins » et vérifier que le Plugin est là. 
6. Cliquer sur « Activate » en dessous de « WP Mail SMTP ». 


## Envoyer des mails 


1. Cliquer sur « Settings » à côté du Plugin 
2. Remplir le champs « Force From Email » 
3. Remplir tous les autres champs
