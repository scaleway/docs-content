<Message type="important">
  This feature is **no longer supported** by AMD and no longer available with the current Instances ranges.
</Message>

[Enterprise Instances](/compute/instances/concepts/#enterprise-instance) rely on the AMD EPYCâ„¢ 7003 series Processor, which provides the SEV feature. AMD SEV helps keep your Instance's data confidential, by encrypting the memory of the virtual machine. SEV does not require any changes to your application software, it is enabled in the hypervisor and in the image of the Enterprise Instance. More details on the SEV feature are available at the [AMD website](https://developer.amd.com/sev/).

You can check that SEV is activated on your Enterprise Instance by entering the following into its command line:

```
# dmesg | grep -i sev
```
<Message type="note">
  - As the encryption is activated at the boot of the Instance, the boot time can take a few seconds more on larger Enterprise Instances.
  - If the Instance is booted into Rescue Mode, SEV is disabled automatically.
</Message>
