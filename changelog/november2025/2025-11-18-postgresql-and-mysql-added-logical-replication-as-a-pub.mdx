Managed PostgreSQL now supports logical replication as a subscriber, available with PostgreSQL 17. Refer to the [Setting up logical replication as a publisher in PostgreSQL](https://www.scaleway.com/en/docs/managed-databases-for-postgresql-and-mysql/api-cli/logical-replication-as-publisher/) documentation page for more information.
