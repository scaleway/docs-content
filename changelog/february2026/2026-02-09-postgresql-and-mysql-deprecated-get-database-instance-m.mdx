title: Get Database Instance metrics API endpoint deprecated
status: deprecated
date: 2026-02-09
category: databases
product: postgresql-and-mysql
---

The [Get Database Instance metrics endpoint](https://www.scaleway.com/en/developers/api/managed-databases-for-postgresql-and-mysql/#path-database-instances-deprecated-get-database-instance-metrics) is now deprecated. It will be removed on March 10th. All Database Instance metrics are available [in Cockpit](/managed-databases-for-postgresql-and-mysql/how-to/monitor-databases-cockpit/) and can be retrieved [via the Cockpit API](https://www.scaleway.com/en/docs/cockpit/reference-content/cockpit-supported-endpoints/#mimir-query-endpoints).
