status: added
date: 2025-09-23

IAM admins can now set up an Organization alias within their Organization. With this alias, a dedicated URL can be built for any member of the Organization to easily login. [Read our documentation](/organizations-and-projects/how-to/set-organization-alias/) for more information on the feature.
