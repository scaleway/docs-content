SCIM user-provisioning is now available for some Organizations. Combined with SAML, SCIM enable user creation, updates and authentication to be managed by an external Identity Provider. [Read our documentation to learn more](https://www.scaleway.com/en/docs/iam/how-to/manage-users-with-scim/)
