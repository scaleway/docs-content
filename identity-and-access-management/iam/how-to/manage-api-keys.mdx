All users are able to view and delete their own API keys as necessary. API keys can also be edited, to change the description or [preferred Project](/identity-and-access-management/iam/api-cli/using-api-key-object-storage/). Instructions for these actions are given below. If you are the Owner of your Organization, or have [sufficient permissions](/identity-and-access-management/iam/reference-content/permission-sets) to do so, you can also view, edit and delete the API keys of other IAM users / applications.

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
</Message>

## How to edit an API key

1. Click **Identity and Access Management (IAM)** from the top-right of your [**Organization Dashboard**](https://console.scaleway.com/organization) in the Scaleway console. You are taken to your [Identity and Access Management dashboard](https://console.scaleway.com/iam/users).
    <Lightbox src="scaleway-iam-dashboard.webp" alt="" />

2. Click the **API Keys** tab. 

3. Click the <Icon name="more" /> icon next to the API key you want to edit, and select **Edit**. The following screen displays:

    <Lightbox src="scaleway-edit-api-key.webp" alt="" />

4. Edit the API key as required:
    - Edit the *description**, and/or
    - Chanage the [preferred Project](/identity-and-access-management/iam/concepts/#preferred-project) for Object Storage

5. Click **Validate** to finish. The API key is edited and you are returned to the list of API keys.

## How to delete an API key 

1. Click **Identity and Access Management (IAM)** from the top-right of your [**Organization Dashboard**](https://console.scaleway.com/organization) in the Scaleway console. You are taken to your [Identity and Access Management dashboard](https://console.scaleway.com/iam/users).
    <Lightbox src="scaleway-iam-dashboard.webp" alt="" />

2. Click the **API Keys** tab. 

3. Click the <Icon name="more" /> icon next to the API key you want to edit, and select **Delete**.

4. Type **DELETE** to confirm, and click **Submit**.

The API key is deleted, and you are returned to the list of current API keys.
