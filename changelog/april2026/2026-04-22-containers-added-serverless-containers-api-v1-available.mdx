The Scaleway Serverless Containers API is now generally available as `v1` (previously `v1beta1`,  which is still available). This stable release includes significant improvements and lays the foundation for upcoming advanced features.

Console, CLI, Terraform, and other tools maintained by Scaleway will be gradually updated.

For details, see [the dedicated v1 documentation](/serverless-containers/reference-content/v1-migration-guide/).
