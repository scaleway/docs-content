This page explains how to create a [version](/identity-and-access-management/secret-manager/concepts/#version) of a secret using the [Scaleway console](https://console.scaleway.com).

<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have [generated an API key](/identity-and-access-management/iam/how-to/create-api-keys/) and enabled the `SecretManagerFullAccess` [permission set](/identity-and-access-management/iam/reference-content/permission-sets/)
  - Your have [created a secret](/identity-and-access-management/secret-manager/how-to/create-secret/)
</Message>

1. Click **Secret Manager** in the **Security and Identity** section of the [Scaleway console](https://console.scaleway.com/) side menu.
2. Select the [region](/identity-and-access-management/secret-manager/concepts/#region) in which to create the version, in the **Region** drop-down.
3. Access the secret you want to add a version to. Your secret's **Overview** tab displays.
4. Click the **Versions** tab.
5. Click **+ Create version**. A pop-up displays.
6. Add your version manually or import it from a file.
7. Click <Icon name="toggle" /> to enable or disable the version.
8. Click **Create version**.
