You can now deploy **Proxmox Virtual Environment (VE) 9** and **Proxmox Backup Server (BS) 4** on compatible Scaleway Dedibox dedicated servers.

Proxmox VE 9, based on Debian 13 Trixie, brings significant improvements including:
* **VM Snapshots on LVM:** Support for snapshots on thick-provisioned LVM storage.
* **Modernized Web Interface:** A revised mobile-friendly interface for easier management on the go.
* **Updated Core:** Integration of Linux Kernel 6.14, QEMU 10.0, and LXC 6.0.

Proxmox BS 4 also introduces key updates, such as S3-compatible object storage as a backup backend and improved ZFS management.

These operating systems are available for new installations directly from the Scaleway console or the Dedibox console for all compatible hardware configurations.
