In the Secret Manager section of the console, you can now select a region to display secrets and folders inside this region. Whenever a secret is created, there is no more need to specify the secret, as it will be created in the region you are currently working on.

Read more about Secret Manager logs in our [dedicated documentation](https://www.scaleway.com/en/docs/identity-and-access-management/secret-manager/concepts/)
