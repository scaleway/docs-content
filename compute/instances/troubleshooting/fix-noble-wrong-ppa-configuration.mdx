The newest Ubuntu 24.04 Noble Numbat Instances created prior to June 19 2024 may have its Scaleway PPA pointing to the staging PPA.<br />

Instances created after June 19th 2024 will not encounter this problem.

## Fix the PPA configuration

You can remediate to this situation by performing the following commands :
```
rm -f /etc/apt/sources.list.d/scaleway-ubuntu-staging-noble.sources
add-apt-repository ppa:scaleway/stable
```
And hit <Enter> when prompted to.
