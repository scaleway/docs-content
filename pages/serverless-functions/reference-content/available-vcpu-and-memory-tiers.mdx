A Serverless Function only works with a strict list of possible resources. When deploying a Serverless Function, make sure to use one of the following pair of memory-CPU. 

|Provision resources for a Serverless Function, in Memory (MB): CPU (mVCPU)|
|---------|
|128: 70|
|256: 140| 
|512: 280|
|1024: 560|
|2048: 1120| 
|3072: 1680| 
|4096: 2240|

<Message type="note">
  This list does not apply to Serverless Containers.
</Message>
