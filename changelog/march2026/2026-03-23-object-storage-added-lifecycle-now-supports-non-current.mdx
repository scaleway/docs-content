title: Lifecycle rules now support non-current versions (Amsterdam, Milan, Warsaw)
status: added
date: 2026-03-23

You can now transition and expire non-current versions of objects in the Scaleway console with automated lifecycle rules. Refer to the [dedicated documentation](/object-storage/how-to/manage-lifecycle-rules/) to fine-tune the lifecycle of your objects.

Support for the Paris region will be available soon.
