### What is included
You can now:

- Track IAM principals (User, Application, Group) creation, edition, deletion, and configuration changes
- Monitor changes in permissions through policy changes
- See all actions performed on Members

Find out more about the IAM supported Audit Trail endpoints in the [dedicated documentation](/audit-trail/reference-content/adt-supported-endpoints/#iam).

### Key benefits

- Full visibility within 2 seconds of action completion
- Complete audit logs with user IDs and timestamps
- Simple filtering by action type and principal
- One-month log retention

Access this feature in the [Audit Trail section](https://account.scaleway.com/?service=elementsaudit-trail/) of the Scaleway console.
