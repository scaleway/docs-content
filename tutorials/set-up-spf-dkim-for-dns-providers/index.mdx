In this tutorial, you will learn how to set up [SPF]() and [DKIM]() records with Transactional Email for other DNS providers such as [GoDaddy](https://www.godaddy.com/), [Cloudflare]() and [Namecheap](). Make sure you retrieve your DKIM and SPF

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have [configured your domain(s) with Transactional Email](/managed-services/transactional-email/how-to/configure-domain-with-transactional-email).
  - You have domain names with GoDaddy, Cloudflare and/or Namecheap
</Message>

## Configure DNS settings for GoDaddy

1. Log in to your [GoDaddy](https://www.godaddy.com/) account.
2. Click **Domains** then click the three dots next to the domain name you want to configure.
3. Click **Manage DNS**. The DNS Management page displays with your existing DNS records.

###  Setting up SPF for GoDaddy

1. Click **Add** in the **DNS Records** section.
2. Select **TXT** in the **Type** drop-down list.
3. Enter "@" in the **Name** section.
4. Copy the following value if you have no other SPF records configured:
    ```
    v=spf1 include:_spf.tem.scw.cloud -all
    ```
    <Message type="note">
    If you have an SPF record configured, insert the following value right before the terminating mechanism (-all) of your record: `include:_spf.tem.scw.cloud`.
    </Message>

5. Paste the value you have copied into the **Value** section.
6. Select **1 Hour** in the **TTL** section.
7. Click the **Save** button.

###  Setting up DKIM for GoDaddy

1. Click **Add** in the **DNS Records** section.
2. Select **TXT** in the **Type** drop-down list.
3. Copy the name that displays in the [Scaleway console]().
    <Lightbox src="scaleway_dkim_name_value.webp" alt="" />
4. Paste it in the **Name** section.
5. Copy the value that displays in the console and paste it in the root zone of your domain on your DNS zone.
    <Lightbox src="scaleway_dkim_record_name_value.webp" alt="" />
6. Paste the value you have copied into the **Value** section.
7. Select **1 Hour** in the **TTL** section.
8. Click the **Save** button.
