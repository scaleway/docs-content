Serverless Jobs API changes from `v1alpha1` to `v1alpha2` with new features:
- `command` field has been upgraded to `startup_command` and `args` for improved flexibility and more use cases
- `reason` field added to give more context about how a job is terminated

For more details about the changes [read our dedicated documentation](https://www.scaleway.com/en/docs/serverless-jobs/reference-content/v1alpha1-to-v1alpha2/).
