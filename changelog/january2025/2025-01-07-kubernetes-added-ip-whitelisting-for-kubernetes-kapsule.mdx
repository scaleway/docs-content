Kubernetes Kapsule users can now restrict cluster access by specifying allowed IPs in CIDR notation. The default configuration `0.0.0.0/0` allows connections from all IPs.  
<Message type="note">
  Kosmos clusters require unrestricted access from Scaleway IP subnets. This security enhancement reduces the risk of unauthorized access. Find more information in the [documentation](https://www.scaleway.com/en/docs/kubernetes/how-to/manage-allowed-ips/).
</Message>
