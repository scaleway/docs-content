The Scaleway FinOps API is now available in beta. This release now allows analysis of costs with a per-resource granularity and fine-grained time granularity, and lays the foundation for upcoming evolutions in FinOps.

For details, see [The dedicated FinOps API Reference documentation](https://www.scaleway.com/en/developers/api/billing_finops).
