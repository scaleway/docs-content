This release upgrades the Kubernetes control plane and node components to version 1.36, including updated networking, storage, and observability components.

### Control plane & add-ons

- Calico: `v3.32.0`
- Cilium: `v1.19.5`
- Kilo: `0.7.0`
- Metrics Server: `v0.8.1`
- CoreDNS: `v1.14.2`
- CSI components: `v0.3.5`
- Cloud Controller Manager (CCM): `v0.36.1`

### Node changelog components

- Linux kernel: `6.8.0-124-generic`
- containerd: `2.3.1`
- CNI plugins: `1.9.1`
- runc: `1.4.2`

### Feature gates available in 1.36

The following feature gates are available and can be enabled as needed:

- `HPAScaleToZero`
- `CPUManagerPolicyAlphaOptions`

For more information, refer to [Scaleway's Kubernetes version support policy](/kubernetes/reference-content/version-support-policy/).
