<Message type="requirement">

  - You have an account and are logged into the [Scaleway Console](https://console.scaleway.com)  - You have [created an Object Storage bucket](/storage/object/how-to/create-a-bucket/)

</Message>

## How to change class from Standard to Glacier

1. Click **Object Storage** in the **Storage** section of the side menu. The list of your buckets displays.
2. Click the bucket name that contains the objects you want to transfer to `GLACIER`. The list of objects stored in the bucket displays.
3. Click the <Icon name="glacier" /> icon next to the object that you want to transfer.
4. Confirm the transfer of the object to the Glacier storage class by clicking on **Send to S3 Glacier**: 
  <Lightbox src="scaleway-send-to-glacier.png" alt="" />

Your object is now being transferred to the `GLACIER` storage class.

## How to change class from Glacier to Standard

1. Click **Object Storage** in the **Storage** section of the side menu. The list of your buckets displays.
2. Click the bucket name that contains the objects you want to restore in `STANDARD` storage class. The list of objects stored in the bucket displays.
3. Click the **Restore icon** next to the object that you want to restore: 
4. Choose the number of days you want to make the object available again in the `STANDARD` storage class and confirm by clicking **Restore object from S3 Glacier**: 
  <Lightbox src="scaleway-restore-from-glacier.png" alt="" />

Your object is restored. 
<Message type="note">

Depending on the size, the restoration process may take up to 6 hours. Your object will remain available in `STANDARD` storage class for the duration you chose during the restore process. It will be transferred automatically back to `GLACIER` once the configured period is over. 

</Message>
