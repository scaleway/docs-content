Scaleway Edge Services' [Web Application Firewall (WAF)](/reference-content/understanding-waf) is now available in the console, in Public Beta.

WAF provides automated protection for web applications against common threats and vulnerabilities, including SQL injection, XSS, remote code execution, and more. It is based on the OWASP Core Rule Set to detect and block malicious requests.
