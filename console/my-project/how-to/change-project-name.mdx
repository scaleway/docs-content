<Message type="requirement">

- You have an account and are logged into the [Scaleway Console](https://console.scaleway.com)
- You have [created a project](/console/my-project/how-to/create-a-project) other than the [default project](/console/my-project/concepts/#default-project)

</Message>

<Message type="important">

The name of the default project cannot be altered. 

</Message>

1. Click **Project Dashboard** in the [Scaleway Console](https://console.scaleway.com) side menu. 

2. Click the **Settings** tab.

3. Click **Change Project Name**.
  <Lightbox src="scaleway-change-project-name.png" alt="" />

  A pop-up with a box to edit the name appears. 
  <Lightbox src="scaleway-change-project-name-pop-up.png" alt="" />

4. Enter the new name of your project.

5. Click **Change Project Name**.
