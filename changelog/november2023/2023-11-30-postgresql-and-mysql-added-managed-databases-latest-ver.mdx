Scaleway Managed Databases now supports PostgreSQL 15.5, 14.10, 13.13, 12.17 and FerretDB 1.14. The MySQL stability has also been improved.

For more information, refer to the [PostgreSQL](https://www.postgresql.org/about/news/postgresql-161-155-1410-1313-1217-and-1122-released-2749/) and [FerretDB](https://blog.ferretdb.io/ferretdb-releases-v1140/) release notes.
