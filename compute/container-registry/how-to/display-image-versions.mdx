Scaleway Container Registry supports image versioning. This allows you to track any changes in your container image versions, and easily revert your deploymet if needed. You can choose which image version you want to pull, and delete any versions that you no longer need.

<Message type="requirement">

- You have an account and are logged into the [Scaleway console](https://console.scaleway.com) 
- You have created a [Container Registry namespace](/compute/container-registry/how-to/create-namespace)

</Message>

1. Click **Container Registry** in the **Tools** section of the [Scaleway Console](https://console.scaleway.com) side menu. The Container Registry namespaces dashboard displays.

    <Lightbox src="scaleway-registry_namespace_list.webp" /> 

2. Click the namespace's name to list its images:
 
    <Lightbox src="scaleway-registry_list_images.webp" />

3. To view pull instructions for a specific version, click <Icon name="more" /> next to the version. Then click **Pull instructions**.

<Message type="tip">

If a specific version is no longer required, click <Icon name="more" /> > **Delete** next to the identifier to delete it.

</Message>
