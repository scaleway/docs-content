Scaleway Block Storage is now HDS-certified, enabling secure and compliant hosting of healthcare data in the Paris region. Refer to the dedicated [documentation](https://www.scaleway.com/en/docs/block-storage/how-to/host-healthcare-data/) and [Shared Responsibility Model](https://www.scaleway.com/en/docs/block-storage/reference-content/storage-shared-responsibility-model/) before storing data with Block Storage as part of an HDS-compliant architecture.
