| Dedibox PRO-9-M       |                                   |
|-----------------------|-----------------------------------|
| **CPU**               | AMD EPYC™ 7272 (12c/24t, 2.9 Ghz) |
| **RAM**               | 128 GB DDR4 ECC                   |
| **Disk**              | 2 x 1 TB NVMe                     |

| **[RPN](/dedibox-network/rpn/concepts/#rpn) private bandwidth** | 1 Gbps (10 Gbps optional)         |
