After [configuring Docker on your local machine](/containers/container-registry/how-to/connect-docker-cli), you can use it to pull images from their library and push them to your Container Registry. This page shows you how to pull, tag, and push a first image to your Scaleway Container Registry namespace, using an Ubuntu image as an example.

<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com) 
  - You have created a [namespace](/containers/container-registry/how-to/create-namespace)
  - You have installed [Docker](https://www.docker.com/) on your local computer
  - You are [logged into your namespace](/containers/container-registry/how-to/connect-docker-cli)
</Message>

1. Open a terminal window on your local computer. 
2. Pull the latest release of the [Ubuntu](https://hub.docker.com/_/ubuntu/) docker image:
    ```
    docker pull ubuntu:latest
    ```
3. Tag the image:
    ```
    docker tag ubuntu:latest rg.fr-par.scw.cloud/mynamespace/ubuntu:latest
    ```
4. Push the image to your Container Registry namespace using docker:
    ```
    docker push rg.fr-par.scw.cloud/mynamespace/ubuntu:latest
    ```

<Message type="important">
  Replace `mynamespace` in the examples above with the name of your namespace.
</Message>
