Dedibox Pro-11 range servers do not currently support KVM-over-IP access.

<Requirements />

- A Dedibox account logged into the [Dedibox console](https://console.online.net)
- A Dedibox Pro-11 range server

## Affected servers
The following server models are affected:
- Dedibox Pro-11 range

## Why KVM-over-IP is not available
KVM-over-IP functionality is currently unavailable on Pro-11 servers due to hardware-specific compatibility constraints with the remote management module.
This limitation is related to the server's baseboard management controller (BMC) configuration and is not caused by user settings or account permissions.

## What this means for you

Without KVM-over-IP, you will not be able to:
- Remotely access the server console during boot
- Troubleshoot OS-level issues without direct access
- Monitor low-level system events via IPMI

## Next steps and roadmap

Our engineering team is actively working to resolve the compatibility issues and enable full KVM-over-IP support for the Pro-11 range. While a public release date is not yet confirmed, this functionality is a priority and will be rolled out as soon as testing and validation are complete.

We will update this page as soon as support becomes available. In the meantime, we recommend using alternative access methods such as SSH.
