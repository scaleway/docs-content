Our Python-API-Framework, that enable you to simply create and deploy Python Serverless Functions using only Python code, now supports local testing with the command `scw-serverless dev app.py`.
For more information and examples check [the project docs](https://serverless-api-project.readthedocs.io/en/latest/)
