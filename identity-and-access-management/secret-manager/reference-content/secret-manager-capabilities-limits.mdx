This page provides information about the capabilities and limits of Scaleway Secret Manager service, depending on user quotas.

Every [Organization](/console/my-account/concepts/#organization) has quotas, which are limits on the number of Scaleway resources they can use. Below is a list of quotas available for Secret Manager.

<Message type="note">
 If you want to increase your quota beyond the values shown on this page, [contact our support team](https://console.scaleway.com/support/create).
</Message>

| Name                                   | [Payment method validated](/console/my-account/how-to/manage-billing-information/#how-to-add-a-credit-card) | [Identity validated](/console/my-account/how-to/verify-identity/) |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| Secrets                                | 100                                                                                                                                         | 250                                                                                               |
| Versions per secret                    | 20                                                                                                                                          | 20                                                                                                |
| Requests to access your secret's value | 300 000                                                                                                                                     | 750 000                                                                                           |
| Maximum size for a secret              | 64kB                                                                                                                                        | 64kB                                                                                              |
