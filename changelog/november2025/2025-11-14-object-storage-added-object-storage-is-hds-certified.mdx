Scaleway Object Storage is now HDS-certified, enabling secure and compliant hosting of healthcare data in the Paris region. Refer to the [dedicated documentation](/object-storage/how-to/host-healthcare-data/) and [Shared Responsibility Model](/object-storage/reference-content/storage-shared-responsibility-model) before storing data with Object Storage as part of an HDS-compliant architecture.
