This release upgrades the Kubernetes control plane and node components to version **1.35**, including updated networking, storage, and observability components.

### Control plane & add-ons

- **Calico:** `v3.31.3`
- **Cilium:** `v1.18.5`
- **Metrics Server:** `v0.8.0`
- **CoreDNS:** `v1.13.2`
- **CSI components:** `v0.3.5`
- **Cloud Controller Manager (CCM):** `v0.35.0`

### Node changelogomponents

- **Linux kernel:** `6.8.0-64-generic`
- **containerd:** `2.2.1`
- **CNI plugins:** `1.9.0`
- **runc:** `1.4.0`

### Feature gates available in 1.35

The following feature gates are available and can be enabled as needed:

- `HPAScaleToZero`
- `CPUManagerPolicyAlphaOptions`
- `MutatingAdmissionPolicy`

For more information, refer to [Scaleway's Kubernetes version support policy](/kubernetes/reference-content/version-support-policy/).
