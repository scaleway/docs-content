[Meta Llama 3.1 8b](/ai-data/managed-inference/reference-content/llama-3.1-8b-instruct/), [Meta Llama 3.1 70b](/ai-data/managed-inference/reference-content/llama-3.1-70b-instruct/) and [Mistral Nemo](/ai-data/managed-inference/reference-content/mistral-nemo-instruct-2407/) are available for deployment on Managed Inference. 

Released July 2024, these models all support a very large context window of up to 128k tokens, particularly useful for RAG applications.
