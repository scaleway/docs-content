If you use compute [Instances](https://www.scaleway.com/en/virtual-instances/) at Scaleway Elements, you are now able to see what is going into the managed dashboard into Observability. You have access to the following key metrics:
- CPU usage
- Disk read/write
- Network in/out
- Network packets in/out
