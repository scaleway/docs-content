title: Kubernetes version 1.32 is now available!
status: added
date: 2025-03-24

Scaleway managed Kubernetes products will support the latest minor version 1.32, Penelope, until Mai 2026.

In addition to patch versions 1.32.0, 1.32.1, 1.32.2, releases 1.31.4, 1.31.5, 1.31.6, and releases 1.30.4 to 1.30.10 are also available. Read our [version support policy](https://www.scaleway.com/en/docs/kubernetes/reference-content/version-support-policy/) for more details on the 14-month support timeline.
