We are pleased to announce that Instance templates are now available. 

An Instance template allows you to save a predefined, immutable server configuration, including Instance type, boot image, volumes, network settings, tags, and cloud-init scripts, so you can repeatedly deploy consistent Instances without manually re-entering parameters.

You can create templates from scratch via the new Templates tab in the console, or save your configuration as a template during the standard Instance creation wizard.
