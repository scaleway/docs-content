  ## Serverless SQL

  ### Serverless SQL Databases supported endpoints

    The following [Serverless SQL Databases](/serverless-sql-databases/) endpoints are supported and will be logged in Audit Trail:

    - CreateDatabase (`POST /serverless-sqldb/v1alpha1/regions/{region}/databases`)
    - UpdateDatabase (`PATCH /serverless-sqldb/v1alpha1/regions/{region}/databases/{database_id}`)
    - DeleteDatabase (`DELETE /serverless-sqldb/v1alpha1/regions/{region}/databases/{database_id}`)
    - RestoreDatabaseFromBackup (`POST /serverless-sqldb/v1alpha1/regions/{region}/databases/{database_id}/restore`)
    - ExportDatabaseBackup (`POST /serverless-sqldb/v1alpha1/regions/{region}/backups/{backup_id}/export`)

</Accordion>
