This datasheet provides a concise overview of the performance, technical features, components, materials, and associated documentation for the different Elastic Metal server offers.

<Message type="note">
* The **Public bandwidth** refers to the designated internet bandwidth for the server, while the **Private bandwidth** is the permitted bandwidth for the server within Private Networks.
  Both public internet and Private Networks use the same physical network interface.
* **Water usage** is only included for zones where values are available; missing zones indicate unavailable data.
</Message>
