## Object Operations

| Feature | Description | Status |
| :------------ | :---- | :- |
|  [DeleteObjects](/storage/object/api-cli/object-operations/#deleteobjects) |  Delete multiple objects in a single request (up to 1000) | <Icon name="validate" /> |
|  [DeleteObject](/storage/object/api-cli/object-operations/#deleteobject) |  Deletes an object | <Icon name="validate" /> |
|  [DeleteObjectTagging](/storage/object/api-cli/object-operations/#deleteobjecttagging) |  Deletes the tag(s) of an object | <Icon name="validate" /> |
|  [GetObject](/storage/object/api-cli/object-operations/#getobject) |  Retrieves an object | <Icon name="validate" /> |
|  [GetObjectAcl](/storage/object/api-cli/object-operations/#getobjectacl) |  Gets the ACL of an object | <Icon name="validate" /> |
|  [GetObjectLegalHold](/storage/object/api-cli/object-lock/#get-object-legal-hold) |  Gets an object's current [Legal Hold configuration](/storage/object/api-cli/object-lock/#how-to-manage-object-legal-hold) | <Icon name="validate" /> | 
|  [GetObjectLockConfiguration](/storage/object/api-cli/object-lock/#get-object-lock-configuration) |  Gets an [object Lock configuration](/storage/object/api-cli/object-lock/) for a bucket. The specified rule will be applied by default to every new object placed in the bucket | <Icon name="validate" /> |
|  [GetObjectRetention](/storage/object/api-cli/object-lock/#get-object-retention) |  Retrieves an object's [retention settings](/storage/object/api-cli/object-lock/#how-to-configure-object-retention) | <Icon name="validate" /> |
|  [GetObjectTagging](/storage/object/api-cli/object-operations/#getobjecttagging) |  Get the tag(s) of an object | <Icon name="validate" /> |
|  [HeadObject](/storage/object/api-cli/object-operations/#headobject) |  Gets object metadata | <Icon name="validate" /> |
|  [Multipart](/storage/object/api-cli/object-operations/#createmultipartupload) | All calls related to multipart uploads, such as Initiate, Complete, Abort, List Parts, List Multipart Uploads and Upload Part, are supported | <Icon name="validate" /> |
|  [OPTIONS object](/storage/object/api-cli/object-operations/#options-object) |  Allows to send a preflight request to trigger an evaluation of the rules that are defined in the CORS configuration | <Icon name="validate" /> |
|  [POST object](/storage/object/api-cli/object-operations/#post-object) |  Adds an object to a bucket by using HTML forms | <Icon name="validate" /> |
|  [PutObject](/storage/object/api-cli/object-operations/#putobject) |  Adds an object to a bucket | <Icon name="validate" /> |
|  [PutObjectAcl](/storage/object/api-cli/object-operations/#putobjectacl) |  Adds an ACL to an object | <Icon name="validate" /> |
|  [PutObjectCopy](/storage/object/api-cli/object-operations/#putobjectcopy) |  Copies an object | <Icon name="validate" /> |
|  [PutObjectTagging](/storage/object/api-cli/object-operations/#putobjecttagging) |  Adds one or several tags to an object | <Icon name="validate" /> |
|  [PutObjectLegalHold](/storage/object/api-cli/object-lock/#put-object-legal-hold) |  Applies a [Legal Hold configuration](/storage/object/api-cli/object-lock/#how-to-manage-object-legal-hold) to the specified object | <Icon name="validate" /> |
|  [PutObjectRetention](/storage/object/api-cli/object-lock/#put-object-retention) |  Places an [object Retention configuration](/storage/object/api-cli/object-lock/#how-to-configure-object-retention) on an object | <Icon name="validate" /> |
| [RestoreObject](/storage/object/api-cli/object-operations/#restoreobject) |  Restores an object from Glacier | <Icon name="validate" /> |
