This page explains how to create [folders](/identity-and-access-management/secret-manager/concepts/#folder) and subfolders using the [Scaleway console](https://console.scaleway.com).

<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have [generated an API key](/identity-and-access-management/iam/how-to/create-api-keys/) and enabled the `SecretManagerFullAccess` [permission set](/identity-and-access-management/iam/reference-content/permission-sets/)
</Message>

## How to create a folder

<Message type="note">
  If you have no resources yet in Secret Manager, you can create a folder when creating your secret.
</Message>

1. Click **Secret Manager** in the **Security and Identity** section of the [Scaleway console](https://console.scaleway.com/) side menu.
2. Click **Create secret**.
3. Choose the region in which you want to store your folder (and secret).
    <Message type="note">
     Your folder and secret will be duplicated in all the Availability Zones of the selected region.
    </Message>
4. Select **Create a new [path](/identity-and-access-management/secret-manager/concepts/#path)** and enter a name for your folder. For example `/my-folder` if you want to create a folder named `my-folder`.

    <Lightbox src="scaleway-create-path.webp" alt="" />
5. Complete the secret creation process and click **Create secret**. Your secret's **Overview** tab displays.
6. Click **< Back to Secrets**. Your folder and its contents display.
7. Click `/` to go back to the root [path](/identity-and-access-management/secret-manager/concepts/#path) where your other resources display, according to the region you have selected in the **Region** drop-down.
     <Lightbox src="scaleway-folder-location.webp" alt="" />

    <Message type="tip">
      Once you have created resources in Secret Manager, you can use the **Region** drop-down to select your desired region, click the **+ Create folder** button, enter a name and click **Create folder**, to create more folders.
    </Message>

## How to create a subfolder

1. Click the folder in which you want to create a subfolder.
2. Click the **+ Create folder** button.
3. Enter a name for your subfolder.
4. Click **Create folder**. Your subfolder displays inside your folder.
