<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have [created an Object Storage bucket](/storage/object/how-to/create-a-bucket/)
</Message>

1. Click **Object Storage** in the **Storage** section of the side menu. A list of your buckets displays.
2. Click the bucket name into which you want to upload your file.
3. Click the grey area to select the files you want to upload into the bucket. Alternatively, you can also drag and drop your files into the grey area.
    <Lightbox src="scaleway-drag-n-drop-bucket.webp" alt="" />
4. Select the files to upload from the file browser tool of your operating system.
    <Message type="important">
      Before uploading your files, make sure the file name is in accordance with the [AWS s3 Object key naming guidelines](https://docs.aws.amazon.com/AmazonS3/latest/dev/UsingMetadata.html#object-key-guidelines).
    </Message>
    <Lightbox src="scaleway-s3-select-files.webp" alt="" />
5. Select the [storage class](/storage/object/concepts/#storage-class) of your object. Scaleway Object Storage allows you to associate storage classes with each of your objects. 
    <Lightbox src="scaleway-select-storage-class.webp" alt="" />

    A list of the objects displays, including the newly uploaded objects.

    <Lightbox src="scaleway-list-of-objects.webp" alt="" />
