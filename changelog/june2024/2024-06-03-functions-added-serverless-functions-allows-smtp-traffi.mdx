Serverless Functions now allows traffic to Scaleway Transactional Email, it was already possible via the API but now directly via SMTP servers ! Perfect way to send emails in your workflows.

Check out our [example project](https://github.com/scaleway/serverless-examples/tree/main/functions/python-tem-smtp-server)
