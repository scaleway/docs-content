Allowed IPs define remote access restrictions for your Database Instance. The default entry `0.0.0.0/0` enables any host to establish a connection.

<Message type="requirement">

- You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
- You have a [Database for PostgreSQL and MySQL](/databases/databases-for-postgresql-and-mysql/concepts/#managed-database)

</Message>

## How to add an IP address

1. Click **Databases** in the **Storage** section of the [Scaleway Console](https://console.scaleway.com) side menu. A list of your Database Instances displays.
2. Click the database name or <Icon name="more" /> > **More info** to access the database information page.
3. Click **Allowed IPs** tab. A list of your allowed IP addresses displays.
4. Click **Add IP**. The IP can be a single IP or an IP block. 
  <Message type="note">

  The IP must be specified in CIDR format, i.e. `198.51.100.135/32` for a single IP or `198.51.100.0/24` for an IP block.

  </Message>
5. Enter a single IP address or a sub-network.
  <Message type="note">

  To restore initial settings and allow connections from all IPs, add the following IP block to the list: `0.0.0.0/0`.

  </Message>

## How to delete an IP address

1. Go to your allowed IP address list.
2. Click <Icon name="delete" />.
3. Click **Delete your IP**.
