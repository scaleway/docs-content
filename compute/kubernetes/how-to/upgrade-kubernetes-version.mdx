<Macro id="iam-requirements" />

<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com) 
  - You have [created](/compute/kubernetes/how-to/create-cluster) a Kubernetes Kapsule cluster running on a Kubernetes version prior to the latest release
</Message>

1. Click **Kubernetes** under **Compute** on the side menu. A list of your Kubernetes Kapsule clusters displays.
2. Click the cluster name you wish to upgrade the Kubernetes version for. The cluster information page displays.
    <Message type="tip">
      Alternatively you can click <Icon name="more" /> > **More info** to display the cluster information page.
    </Message>
3. Click **Upgrade** next to the **Kubernetes version** of your cluster. A pop-up displays.
    <Lightbox src="scaleway-kubernetes-cluster-upgrade.webp" alt="" />
4. Select the Kubernetes version you want to upgrade to from the drop-down list. Tick the checkbox **Upgrade cluster node pools as well** if you want also upgrade the Kubernetes version on the node pools of your cluster. 
    <Lightbox src="scaleway-kubernetes-version-select.webp" alt="" />
        <Message type="important">
      Be careful when upgrading the Kubernetes versions of your node pools, as it may lead to data loss on data stored locally on any node.
    </Message>
 5. Click **Upgrade**. 
    <Message type="note">
      It is not possible to downgrade your Kubernetes version once it has been upgraded. 
    </Message>
