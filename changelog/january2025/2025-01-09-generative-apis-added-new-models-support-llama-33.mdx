[Llama 3.3 70B](https://www.scaleway.com/en/docs/ai-data/generative-apis/reference-content/supported-models/#request-a-model) is now available on Generative APIs.

Llama 3.3 is a fine-tune of the Llama 3.1 70b model which was designed to approach the performance of Llama 3.1 405B on some applications.
