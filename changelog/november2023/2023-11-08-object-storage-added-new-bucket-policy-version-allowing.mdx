With version 2023-04-17 of bucket policies, you can grant users and applications granular permissions to specific buckets and objects.
Check out our documentation to discover this new [bucket policy version](https://www.scaleway.com/en/docs/storage/object/api-cli/bucket-policy/) , and learn [how to combine IAM and bucket policies](https://www.scaleway.com/en/docs/storage/object/api-cli/combining-iam-and-object-storage/) to set up granular access to Object Storage!
