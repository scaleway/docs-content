There are different ways to deploy Serverless Containers and this document demonstrates and compares the most popular ways and tools to do it.

## Scaleway console

The [Scaleway Console](https://console.scaleway.com/) is a ready-to-go tool to deploy your Ccntainers.

Console let you deploy containers pushed in the Scaleway Container Registry but also from public repositories.

## Serverless framework

Serverless Framework is a dedicated tool to deploy Serverless Functions and Containers.

It makes it really simple with minimum effort to deploy your Dockerfile a single command `serverless deploy`.
Container configuration is done in a single file and it's easy to track in version control systems.

Install, examples, usage and advanced documentation is avaiable in the [Serverless Scaleway Plugin repository](https://github.com/scaleway/serverless-scaleway-functions).

Serverless Framework can be used in many use cases such as CI/CD, production deployment, 
multiple project deployments with [serverless-compose](https://www.serverless.com/framework/docs/guides/compose).

## Terraform

Serverless Containers integrates easily with your infrastrucure as code, we provide a
[Terraform integration for Coontainers](https://registry.terraform.io/providers/scaleway/scaleway/latest/docs/resources/container).

## API

[Scaleway HTTP API](https://developers.scaleway.com/en/products/containers/api/) allows managing Serverless Functions and Containers
with HTTP calls.

API Container management can be useful for integration in automated tasks, continuous integrations and limited environments.

## CLI

[Scaleway CLI](https://github.com/scaleway/scaleway-cli) is a simple command line interface tool that allows you to drive your Serverless
Functions and Containers.

Install instructions and documentation is available on the [Scaleway CLI repository](https://github.com/scaleway/scaleway-cli).

## Scaleway SDKs

Scaleway SDKs allow you to manage your resources directly from your favorite languages.

Here are our available SDKs:

- [Go SDK](https://github.com/scaleway/scaleway-sdk-go)
- [JS SDK](https://github.com/scaleway/scaleway-sdk-js)
- [Python SDK](https://github.com/scaleway/scaleway-sdk-python)

SDKs documentation is directly in the repositories.
