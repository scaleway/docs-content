You can now connect your Serverless Containers with Private Network, allowing secured access between Serverless to resources inside the same Private Network like databases. 

[Read more](https://www.scaleway.com/en/docs/serverless-containers/how-to/use-private-networks/) about attaching a Private Network to a Serverless Container.
