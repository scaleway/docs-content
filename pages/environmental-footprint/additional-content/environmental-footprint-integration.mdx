This article lists which Scaleway products integrate the [Environmental Footprint calculator](/environmental-footprint/additional-content/environmental-footprint-calculator).

## Available product integration

The following products integrate the Environmental Footprint calculator:
  - Apple silicon
  - Block Storage
  - CPU & GPU Instances
  - Dedibox (data available in the console, but not yet included in the monthly report)
  - Elastic Metal
  - Generative APIs
  - Kubernetes
  - Load Balancers
  - MongoDB 
  - Object Storage
  - PostgreSQL and MySQL
  - Redis
