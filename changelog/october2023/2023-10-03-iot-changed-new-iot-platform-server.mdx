The Certificate Authority used by all MQTT TLS connections to IoT Hubs has been updated.

As described in our [Certificate Authority update documentation](/managed-services/iot-hub/reference-content/certificate/), the `migration.iot.fr-par.scw.cloud` endpoint is no longer valid.
