[Autoscaling](https://www.scaleway.com/en/docs/serverless-containers/concepts/#autoscaling) now supports new rules: CPU and RAM percentage. 

This feature allows you to automatically adjust the CPU and RAM resources allocated to your Serverless Containers based on workload demands.
