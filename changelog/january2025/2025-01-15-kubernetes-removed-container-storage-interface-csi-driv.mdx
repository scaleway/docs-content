The Scaleway Block Volume Container Storage Interface (CSI) driver v.0.2 will no longer be supported by 31/01/2025.

Users can upgrade to the latest CSI driver (v0.3) now.

Learn more [here](https://www.scaleway.com/en/docs/containers/kubernetes/api-cli/managing-storage).
