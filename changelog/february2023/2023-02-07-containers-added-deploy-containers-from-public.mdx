You can now deploy container images stored in public registries such as Docker Hub or your personal container registry to Serverless Containers. 
Simply write the image address (for example `wordpress:latest` to deploy the latest Wordpress image from [Docker Hub](https://hub.docker.com/) in the [Scaleway console](https://console.scaleway.com), Scaleway's API, Terraform or Serverless Framework
