To learn more about Frontends and Backends, refer to the [Load Balancers concepts page.](/network/load-balancer/concepts)

<Message type="requirement">

- You have an account and are logged into the [Scaleway Console](https://console.scaleway.com) 
- You have [created a Load Balancer](/network/load-balancer/how-to/create-load-balancer)

</Message>

## How to add frontends

1. Click **Load Balancers** in the **Network** section of the [Scaleway Console](https://console.scaleway.com) side menu. The Load Balancer dashboard displays. 

2. Click the Load Balancer you want to add frontends to. The Load Balancer information page displays. 

3. Click the **Frontends** tab.

4. Click **Add Frontend**.
  <Lightbox src="assets/scaleway-add-frontends.png" />

5. Enter a **name**, **port** and **protocol** for your new frontend. 

6. Choose between **creating a new backend** or **using an existing backend**.
    - If creating a new backend, enter a name, port, protocol, proxy, health check type, sticky session configuration and server IP
    - If using an existing backend,  click **Use an existing Backend** and select the backend of your choice from the dropdown list.

7. Click **Create Frontend**.

## How to edit frontends

1. Click **Load Balancer** in the **Network** section of the [Scaleway Console](https://console.scaleway.com) side menu. The Load Balancer dashboard displays. 

2. Click the Load Balancer you want to edit. The Load Balancer information page displays. 

3. Click the **Frontends** tab.

4. Click <Icon name="more" /> next to the frontend you want to edit

5. Click **Edit**.

6. Make your edits. You can change:
    - Frontend name
    - Backend name
    - Port
    - Associated SSL Certificate(s)

4. Click **Edit Frontend**.

## How to add backends

1. Click **Load Balancer** in the **Network** section of the [Scaleway Console](https://console.scaleway.com) side menu. The Load Balancer dashboard displays. 

2. Click the Load Balancer you want to add backends to. The Load Balancer information page displays. 

3. Click the **Backends** tab.

4. Click **Add Backend**.
  <Lightbox src="assets/scaleway-add-backends.png" />

5. Enter a name, port, protocol, proxy, health check type, sticky session configuration and server IP for your new backend. 

6. Click **Create Backend**.

## How to edit backends

1. Click **Load Balancer** in the **Network** section of the [Scaleway Console](https://console.scaleway.com) side menu. The Load Balancer dashboard displays. 

2. Click the Load Balancer you want to edit. The Load Balancer information page displays. 

3. Click the **Backends** tab.

4. Click <Icon name="more" /> next to the backend you want to edit.

5. Click **Edit**.

6. Make your edits. 
  <Lightbox src="assets/scaleway-edit-backends.png" />

7. Once your edits are done, click <Icon name="validate" />.
