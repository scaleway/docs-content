---
title: Versioning and Object Lock are now fully supported in the Scaleway console
status: added
date: 2026-05-20
category: storage
product: object-storage
---

Versioning lets you prevent accidental object alteration, preserve object history, search for specific object versions, and manage delete markers directly from the Scaleway console.

Object Lock protects sensitive assets against ransomware and unauthorized changes through a guided interface that reduces the risk of misconfiguration.

Refer to our dedicated [versioning](/object-storage/use-bucket-versioning/) and [object lock](/object-storage/how-to/use-object-lock/) documentation pages to start protecting your data.
