The Scaleway FinOps API is now available in Beta. This release allows analysis of costs with a per-resource granularity and fine-grained time granularity, and it also lays the foundation for upcoming evolutions in FinOps.

For details, see [The dedicated FinOps API Reference documentation](https://www.scaleway.com/en/developers/api/billing_finops).
