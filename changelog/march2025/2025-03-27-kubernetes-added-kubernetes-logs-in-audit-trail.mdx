Kubernetes actions are now visible in Audit Trail, providing comprehensive security monitoring and compliance tracking.

### What's included:
* Track cluster creation, editing, deletion, and configuration changes
* Monitor node pool and node operations
* Log KubeConfig downloads and resets
* Track application deployments and changes

### Key benefits:
* Full visibility within 2 seconds of action completion
* Complete audit logs with user IDs, timestamps
* Simple filtering by action type and resource
* One-month log retention

Access this feature in the [Audit Trail section](https://account.scaleway.com/?service=elementsaudit-trail/) of the Scaleway console.
