<Macro id="iam-requirements" />

<Message type="requirement">
    - You have an account and are logged into the [Scaleway console](https://console.scaleway.com/)
</Message>

1. Click **IPFS Naming** in the **Labs** section of the [Scaleway console](https://console.scaleway.com) side menu. The IPFS Naming section displays.
2. Click **Create a Key** to create a [key](/labs/ipfs-naming/concepts/#name), which is the storage area for a set of pins.
<Lightbox src="create-ipfs-naming.webp" alt="" />
3. Choose a region for the keys.
<Lightbox src="choose-region-naming.webp" alt="" />
4. Enter a description for the keys.

<Lightbox src="choose-description-naming.webp" alt="" />

5. Click **Import Key** to display the new field for importing your private key.
<Lightbox src="import-key-naming.webp" alt="" />

6. Click on **Download key** and choose the file containing your private key.
<Message type="note">
    - You can used `ipfs key export $NAME_KEY` to generate file
</Message>

7. Enter a value for the keys.
<Message type="requirement">
    - The value must be an IPNS key or a CID
</Message>

8. Click **Create a Name**.
