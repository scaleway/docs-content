It is now possible to configure a lifecycle rule from the Console to automatically abort your uncomplete multipart uploads after a chosen number of days. Learn here how to start configuring lifecycle rules in Console: https://www.scaleway.com/en/docs/storage/object/how-to/manage-lifecycle-rules/
